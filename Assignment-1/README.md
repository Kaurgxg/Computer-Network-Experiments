# Experiment 1 -- Network Topologies

## How to Open and Test

1.  Open Cisco Packet Tracer.
2.  Load the file `exp1_topologies.pkt`.
3.  Verify IP configuration on all PCs.
4.  Open Command Prompt on any PC.
5.  Use ping command to test connectivity.

Example:

    ping 192.168.x.x

6.  Switch to Simulation Mode to observe ICMP packet flow.
7.  For failure testing, disconnect one link in the ring topology and
    re-test connectivity.
8. Also I've deleted the earlier Star and bus topologies in which the original practical was done as I modified the topology as per the task. But at last,I again made the topologies using new end devices and network devices.
